News

2015

  • 2015-05-19: Purchased two WDR4300s, flashed with [Commotion] and initial testing of concept is undertaken
  • 2015-05-23: Some preliminary tests of RF propagation across Corbin Reserve. Found that 5GHz goes about 200m, and 2.4GHz can cover most of the reserve as long as line of sight is available to the AP. This is a good result considering this was only using the included antennas and a laptop
  • 2015-05-24: Registered corbinmesh.net and corbinmesh.com, and created this site
  • 2015-05-24: Created contact@corbinmesh.net email address
  • 2015-05-25: We have a twitter account: @corbinmesh
  • 2015-05-26: Submitted corbinmesh.net to the Google Site Indexer
  • 2015-05-27: Collecting information about the NanoStation and PicoStation. Started collecting data sheets for supported models
  • 2015-05-28: Collecting information about the Bullet and Rocket
  • 2015-05-29: Site redesign to flatten links and collate pages to make more mobile friendly
  • 2015-05-30: Collecting images for [setting up a node]
  • 2015-06-04: Updated the hub-and-spoke and mesh diagrams in the mesh section
  • 2015-06-14: Temporary node up and running, very limited propagation and no internet access at this stage. Some configuration may change
  • 2015-06-22: Configured cloudflare for our site, mostly to protect us from @davidmarsh randomly unplugging the server
  • 2015-06-26: Replaced png images on the main CorbinMesh page with svgs, created our own community mesh graphic, removed static map images and instead used an openstreetmap frame. In future this may be changed back as hosting inside the mesh may break this
  • 2015-06-29: Changed openstreetmap for static map, make running off-internet simpler and just addeded an external link to openstreetmap
